
# AI
It's just a simple AI chatbot Docker container using OpenAI GPT-3 and flask (Python) 

## Setup

1. Install Docker Desktop

2. Build Dockerfile 

  ```bash
   $ docker build -t my-app .      
   ```

3. Run Dockerfile

```bash
$ docker run -p 5000:5000 my-app    
   ```
